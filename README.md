## 📚 FYUGP Minor C Practicals – Introduction to Data Science

This repository contains the **practical implementations for the course _“Introduction to Data Science”_ offered as Minor C under Calicut University FYUGP**.

The repository focuses on **hands-on implementation of core data science and machine learning algorithms** using Python and commonly used data science libraries.

Students work with **real-world datasets** and implement fundamental algorithms used in **classification, regression, clustering, and text processing**.

---

# 🎯 Scope of the Repository

The main objectives of this repository are:

- Implement **core data science workflows**
- Understand **data preprocessing and feature engineering**
- Build **machine learning models**
- Evaluate models using **standard performance metrics**
- Work with **real-world datasets**
- Perform **basic Natural Language Processing tasks**

---

# 🧠 Topics Covered

This repository includes practical implementations of the following concepts:

- Binary Classification
- Linear Regression
- k-Nearest Neighbors (k-NN)
- k-Means Clustering
- Spam Detection using NLP
- Data preprocessing and feature scaling
- Model evaluation and performance metrics

---

# 📂 Practical Components

## 1️⃣ Binary Classification – Credit Card Fraud Detection

**Objective**

Build a binary classification model to detect **fraudulent credit card transactions**.

**Dataset**

Credit Card Fraud Detection Dataset

Dataset Download URL:  
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

**Key Concepts**

- Handling **highly imbalanced datasets**
- Data preprocessing and **feature scaling**
- **Train-test split**
- Training classification models such as:
  - Logistic Regression
  - k-NN
  - Decision Trees
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
  - Confusion Matrix

---

## 2️⃣ Linear Regression

**Objective**

Implement **Linear Regression** to model and predict a **continuous target variable** such as:

- House price prediction
- Salary prediction
- Student marks prediction

**Key Concepts**

- Relationship between **independent and dependent variables**
- Fitting a **regression line**
- Interpreting regression coefficients
- Model performance metrics:
  - MAE (Mean Absolute Error)
  - MSE (Mean Squared Error)
  - RMSE (Root Mean Squared Error)
  - R² Score

---

## 3️⃣ k-Nearest Neighbors (k-NN)

**Objective**

Use **k-NN algorithm** for classification tasks based on **similarity between data points**.

**Key Concepts**

- Distance metrics:
  - Euclidean distance
- Choosing optimal **k value**
- Effect of **feature scaling**
- Model evaluation using:
  - Accuracy
  - Confusion Matrix

---

## 4️⃣ k-Means Clustering

**Objective**

Apply **k-Means clustering** for **unsupervised learning** and grouping similar data points into clusters.

**Key Concepts**

- Cluster centroids
- Intra-cluster vs inter-cluster distances
- Determining number of clusters using:
  - **Elbow Method**
- Visualizing clusters using plots

---

## 5️⃣ Spam Filtering (Text Classification)

**Objective**

Build a **spam detection model** that classifies messages as:

- Spam
- Ham (Not Spam)

**Key Concepts**

Text preprocessing techniques:

- Tokenization
- Stopword removal
- Stemming / Lemmatization

Feature extraction methods:

- Bag-of-Words
- TF-IDF

Classification models:

- Naive Bayes
- Logistic Regression

Evaluation metrics:

- Accuracy
- Precision
- Recall
- F1-score
